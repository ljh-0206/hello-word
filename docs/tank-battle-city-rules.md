# 坦克大战 (Battle City) 游戏规则

## 一、游戏概述

红白机经典坦克大战的 Web 复刻版，Canvas 像素风格渲染。

| 项目 | 说明 |
|------|------|
| 画布 | 640 × 480 px |
| 网格 | 20 × 15，每格 32px |
| 帧率 | 60fps (requestAnimationFrame) |

---

## 二、地图元素

| 元素 | 符号 | 说明 |
|------|------|------|
| 空地 | TILE_EMPTY (0) | 坦克和子弹可通行 |
| 砖墙 | TILE_BRICK (1) | 可被子弹摧毁 |
| 钢墙 | TILE_STEEL (2) | 不可摧毁，子弹击中后消失 |
| 基地 | TILE_BASE (3) | 鹰旗，被毁则游戏失败 |
| 森林 | TILE_FOREST (4) | 坦克和子弹可通行，渲染在坦克上方实现视觉遮挡 |
| 水体 | TILE_WATER (5) | 坦克不可通行，子弹可飞过 |

### 基地保护

基地（老鹰）位于最底部中央，由三面砖墙保护（上方、左方、右方），玩家出生在左下区域。

## 三、玩家规则

### 3.1 操作

| 按键 | 动作 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格 | 射击 |
| Enter | 重新开始 / 进入下一关 |
| ESC | 退出游戏 |

### 3.2 属性

| 属性 | 默认值 | 说明 |
|------|--------|------|
| 命数 | 3 | 显示在右上角 |
| 移动速度 | 2px/frame | 固定 |
| 射击冷却 | 15 frames | 等级 3 时缩短为 10 |
| 火力等级 | 1 | 最高 3 级 |
| 复活无敌 | 90 frames | 死亡重生后短暂无敌 |

### 3.3 火力升级

| 等级 | 效果 |
|------|------|
| 1 | 单发普通子弹 |
| 2 | 单发 + 炮管标记 |
| 3 | 三发散射（前方 + 两侧偏移） |

---

## 四、敌方规则

### 4.1 敌人生成

- 顶部三个固定刷新点：（0,0）、（0,9）、（0,18）
- 生成间隔约 2.5 秒
- 场上同时最多 3 个活跃敌人
- 出生后有 120 frames 闪烁无敌

### 4.2 敌人类型

| 类型 | 颜色 | 生命 | 速度 | 分值 | 特征 |
|------|------|------|------|------|------|
| 普通 | 红色 | 1 | 2 | 100 | 基础单位 |
| 快速 | 绿色 | 1 | 4 | 100 | 移动快，射击频率高 |
| 重型 | 紫色 | 3 | 1 | 300 | 血厚攻慢，装甲外观 |

### 4.3 敌人 AI

- 每 1~3 秒尝试变更方向
- 遇到障碍物时立即重新寻找方向
- 射击间隔：普通/快速 0.75~1.5 秒，重型 1.5 秒
- 不会主动追击玩家（随机行走）

### 4.4 波次递增

| 波次 | 敌人数 | 重型出现 |
|------|--------|---------|
| 1 | 6 | 否 |
| 2 | 9 | 概率出现 |
| 3 | 12 | 概率出现 |
| n | 6 + n×3 | 随波次增加 |

## 五、边界与碰撞

### 5.1 碰撞类型

| 碰撞对 | 处理方式 |
|--------|---------|
| 坦克 ↔ 画布边缘 | 被 clamped 限制在边界内 |
| 坦克 ↔ 砖墙/钢墙/基地/水体 | 不可通行 |
| 坦克 ↔ 森林 | 可通行，视觉上被森林遮挡 |
| 坦克 ↔ 坦克（玩家与敌人） | 不可重叠，实时碰撞检测 |
| 坦克 ↔ 坦克（敌人之间） | 不可重叠，实时碰撞检测 |
| 子弹 ↔ 砖墙 | 砖墙被摧毁，子弹消失 |
| 子弹 ↔ 钢墙 | 子弹消失，钢墙不受影响 |
| 子弹 ↔ 基地 | 基地被摧毁 → Game Over |
| 子弹 ↔ 坦克（玩家） | 坦克死亡，命数 -1 |
| 子弹 ↔ 坦克（敌人） | 敌人 HP -1，HP 归零时击毁 |
| 子弹 ↔ 子弹 | 无交互（可穿透） |
| 子弹 ↔ 水体/森林 | 子弹可通过 |

### 5.2 坦克碰撞检测算法

```
canMoveTo(targetPos, allTanks):
  1. 检查四角所在的 tile（任何非空地/森林 → 不可通行）
  2. 检查是否与任何活着的坦克矩形重叠
  3. 两步均通过 → 可移动
```

## 六、道具系统

| 道具 | 图标 | 效果 |
|------|------|------|
| ★ 火力升级 | 黄色星 | 火力等级 +1（最高 3） |
| ◈ 护盾 | 蓝色菱形 | 300 frames 无敌 + 护盾视觉 |
| ● 炸弹 | 红色圆 | 清除屏幕上所有敌人 |

- 每击毁 4 个敌人掉落一个道具
- 同时仅存在一个道具
- 道具持续 600 frames（约 10 秒）后消失
- 玩家坦克与道具矩形重叠时自动拾取

---

## 七、胜负条件

### 胜利
消灭当前波次所有敌人 → 显示 "STAGE CLEAR" → 按 Enter 进入下一关

### 失败（任一触发）
1. 基地（鹰旗）被子弹击中 → 立即 Game Over
2. 玩家命数归零 → Game Over

### 游戏结束操作
- Game Over 时按 Enter → 重新从第 1 波开始
- Stage Clear 时按 Enter → 进入下一波
- 任何时候按 ESC → 退出到主页

---

## 八、游戏流程图

```
开始游戏
  ↓
出生（90 frames 无敌）
  ↓
敌人从顶部生成（有闪烁无敌）
  ↓
玩家移动/射击 ↔ 敌人移动/射击
  ↓
摧毁敌人 → 累计击杀数 → 掉落道具
  ↓
敌人全灭？ → Yes → STAGE CLEAR → 下一关
   No
  ↓
基地被毁 或 命数归零？ → Yes → GAME OVER
   No
  ↓
继续战斗
```
