# DeepSeek 项目成果展示

基于 DeepSeek 能力打造的项目成果展示站。

## 项目列表

- 🎮 **坦克大战** — 经典红白机游戏复刻，基于 Canvas 实现完整游戏逻辑
- 更多项目即将上线...

## 技术栈

- Vue 3 (Composition API + `<script setup>`)
- Vite 8
- Canvas (游戏渲染)

## 本地运行

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

Cloudflare Pages 自动部署，推送 `master` 分支即可触发。
